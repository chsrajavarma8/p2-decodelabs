# Data Classification Using AI

## Objective

Build a machine learning classification model using the Iris Dataset.

## Technologies Used

- Python
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Algorithm

K-Nearest Neighbors (KNN)

## Workflow

1. Load Iris Dataset
2. Split Data into Training and Testing Sets
3. Apply Feature Scaling
4. Train KNN Model
5. Predict Test Data
6. Evaluate using:
   - Accuracy
   - Confusion Matrix
   - F1 Score

## Output

The model successfully classifies Iris flower species:

- Setosa
- Versicolor
- Virginica

## Conclusion

KNN successfully classifies Iris flower species with high accuracy and demonstrates the fundamentals of supervised learning.
